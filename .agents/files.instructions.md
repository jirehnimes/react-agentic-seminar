1. Directory and file names should be in kebab-case (lowercase letters and hyphens).
2. All components should follow this structure:
```
src/components/$component-path/$component-name/
  - index.ts
  - $component-name.tsx
  - $component-name.test.tsx
  - $component-name.hook.ts
  - $component-name.hook.test.ts
```
3. All components default files are:
  - index.ts
  - $component-name.tsx
  - $component-name.test.tsx
